飞书配置
配置说明：配置飞书应用信息后，可以将抓取的数据同步到飞书多维表格中。

获取配置信息步骤：
  1. 访问 飞书开放平台 创建应用
  2. 在应用管理页面获取 App ID 和 App Secret
  3. 创建或打开飞书多维表格，从URL中获取 文档 Token
  4. 在多维表格中获取具体数据表的 表格 ID
  5. 确保应用有多维表格的读写权限
详细教程请参考飞书开放平台文档

飞书多维表格字段说明

系统将自动映射以下字段到飞书多维表格：

  • 推文原文内容 - 推文的完整文本内容
  • 发布时间 - 推文的原始发布时间
  • 作者（账号） - 推文作者的用户名
  • 推文链接 - 推文的直接链接
  • 话题标签 - 推文中的所有hashtag
  • 类型标签 - 自动分类：搞钱、投放、副业干货、情绪类
  • 收藏数、点赞数、转发数 - 推文的互动数据
  • 创建时间 - 数据抓取时间
